Проект создан в рамках хакатона Skillfactory. Данный скрипт создаёт в google-таблице новые листы с анализом необходимой информации.

Для работы на Windows, необходимо скачать стабильную версию Python: https://www.python.org/downloads/release/python-3114/
А так же установить библиотеки, указанные в файле requirements.txt

Запуск через командную строку: ./python3 script.py, либо файл SF_script.exe
